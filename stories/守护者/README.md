# 守护者v2 / 灯塔守护者的谜团

一个基于 [ink](https://www.inklestudios.com/ink) 的交互式叙事游戏项目。

## 项目简介

一个悬疑/奇幻类型的互动小说，玩家扮演一名接替前任灯塔守护者的新任守护者，在暴风雨夜抵达灯塔后，通过探索、选择和调查，揭开前任守护者埃里克留下的谜团。

## 故事特性

- **标签系统**：使用 Ink 标签标记关键节点与选项
- **LIST 随机**：利用 `LIST_RANDOM` 随机生成到达时的环境细节（雨、雷、风）
- **条件测试**：基于变量和状态的分支逻辑
- **多分支结局**：不同选择导向不同故事走向
- **纯节点跳转**：结构化的 `-> knot` 叙事流

## 技术栈

- **引擎**：inkjs（Ink 的 JavaScript 运行时）
- **前端**：原生 HTML5 / CSS3 / JavaScript
- **样式**：响应式布局，支持亮色/暗色主题切换

## 文件结构

| 文件 | 说明 |
|---|---|
| `index.html` | 入口页面，包含故事容器与控制按钮 |
| `ink.js` | inkjs 引擎库（已压缩） |
| `main.js` | 故事渲染、存档/读档、主题切换逻辑 |
| `style.css` | 页面样式与主题定义 |
| `守护者v2.ink` | Ink 故事源文件 |
| `守护者v2.js` | 由 Ink 源文件编译生成的 JSON 故事数据 |

## 运行方式

直接在浏览器中打开 `index.html` 即可运行。项目为纯静态页面，无需构建工具或服务器环境。

## 操作说明

- **restart**：从头重新开始故事
- **save**：保存当前进度
- **load**：加载已保存的进度
- **theme**：切换亮色/暗色主题
