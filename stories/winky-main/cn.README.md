# 温奇 

## 摘要 
发布你自己的互动故事！该网页模板适用于[Ink脚本语言和编辑器]（https://www.inklestudios.com/ink/）。 

这里有示例：[https://winky.vellandi.net]（https://winky.vellandi.net） 

## 许可证 

这个模板由马里奥·维兰迪为作者杰弗里·戈尔登及其出版物《冒险小吃》（https://www.adventuresnack.com/）制作。
 虽然该模板免费使用，但文件“story.js”中的故事及所用图片版权归Geoffrey Golden和Adventure Snack所有。 
要创作自己的故事，可以用这个模板作为基础，但必须使用新的故事文件，最好使用你自己的字体和图片。 

## 快速启动 

1. [下载并安装墨水编辑器]（https://github.com/inkle/inky/releases） 

2. [学习如何用墨水写故事]（https://www.inklestudios.com/ink/web-tutorial/） 

3. 在Inky编辑器中写故事并导出为.js文件 

4. 用你自己的故事文件替换“story.js” 

5. 删除文件夹“img”中的图片，并用你自己的（如适用）替换 


6. 编辑“index.html”以更改内容<\标题\><\h1\>内容

7. 使用你自己的自建或谷歌字体，通过“/assets/fonts.css”，并编辑“/styles/variables.css”来更改字体变量

8. 删除/修改你自定义域名的CNAME文件

9. 在浏览器中打开“index.html”，享受你的故事！

10. 将你的故事托管在GitHub Pages上。


## Styling Images and Choices

Using the Inky Editor, you can add CSS classes to images and choices using the following format:

'# IMAGE: src&class=class1,class2,etc...'  
'# IMAGE: img/mustang.jpg&class=block'

'Choice text &class=class1,class2,etc...'  
'Enter the Graveyard &class=strike'
